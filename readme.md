## nodejs 용 opencv 예제(opencv4nodejs)

### 설치

```bash
npm install opencv4nodejs --save
```
node4opencv 는 자동으로 opencv 를 설치해준다.  
그러므로 별도의 다른 설치가 필요없다.

### 기타 

es6 문법을 지원하는 mjs 파일 실행시키는 방법  
```bash
node --experimental-modules 6.mjs
```
